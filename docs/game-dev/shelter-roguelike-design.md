# 게임 기획서 - 메인 시스템 v1.0

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| **장르** | 로그라이크 액션 + 거점 시뮬레이션 |
| **플레이 인원** | 4인 코옵 (솔로 가능) |
| **핵심 레퍼런스** | 하데스(던전), 명일방주(시뮬레이션) |

---

## 2. 공간 구조

| 구분 | 셀터 (아웃게임) | 외부 세계 (인게임) |
|------|----------------|-------------------|
| **성격** | 안전한 휴식 공간 | 위험한 전투 공간 |
| **컨셉** | 마이 셀터하우스 | 던전/필드/탐험 |
| **활동** | 건물 건설, 자동 생산 | 로그라이크 전투, 재료 수급 |
| **코옵** | X (개인 관리) | O (4인 협력) |

---

## 3. 재화 순환 시스템

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│         [던전]                      [셀터]              │
│            │                           │                │
│            │  건물 건설/업그레이드     │                │
│            │  재료                     │                │
│            ├──────────────────────────►│                │
│            │                           │                │
│            │  유니크 아이템            │                │
│            ├──────────────────────────►│                │
│            │                           │                │
│            │      캐릭터 강화 재화     │                │
│            │◄──────────────────────────┤                │
│            │      (자동 생산)          │                │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**핵심: 서로 반대편의 성장을 도와주는 구조**

| 재화 유형 | 획득처 | 사용처 |
|-----------|--------|--------|
| 건물 건설 재료 | 던전 | 셀터 시설 확장 |
| 캐릭터 강화 재화 | 셀터 (자동) | 캐릭터 육성 |
| 유니크 재료 | 던전 보스 | 특수 장비/고급 시설 |

## 4. 시뮬레이션 시스템

### 설계 원칙

- **소프트함**: 복잡한 조작 없음
- **자동화**: 유닛이 알아서 생산
- **유저 개입 최소화**: 건물 관련만

### 유저 조작 범위

| 하는 것 | 안 하는 것 |
|--------|-----------|
| 건물 건설 | 직접 재배/채집 |
| 건물 업그레이드 | 생산 조작 |
| | 타이밍 맞추기 |

### 시설 구성

| 시설 | 생산물 |
|------|--------|
| 제련소 | 장비 재료 |
| 연구실 | 스킬 포인트 |
| 재배실 | 소모품 (회복/버프) |
| 훈련장 | 경험치 |
| 창고 | 보관 한도 증가 |

### 깊이 요소 (복잡도 없이)

- 시설 레벨업 → 생산량 증가
- 유닛 등급 → 효율 차이
- 보스 드랍 → 고급 시설 해금

## 5. 플레이 사이클

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   1. 던전 입장 (4인 코옵)                               │
│              ↓                                          │
│   2. 로그라이크 전투 → 재료/유니크 획득                 │
│              ↓                                          │
│   3. 셀터 복귀 → 자동 정산 (강화 재화 수령)             │
│              ↓                                          │
│   4. 건물 건설/업그레이드 (선택)                        │
│              ↓                                          │
│   5. 캐릭터 강화 (선택)                                 │
│              ↓                                          │
│   6. 다음 던전 출발                                     │
│                                                         │
└─────────────────────────────────────────────────────────┘
```
