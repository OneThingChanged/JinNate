# Ch.17 실시간 레이 트레이싱

실시간 레이 트레이싱의 기초 이론과 구현을 분석합니다.

---

## 개요

레이 트레이싱은 광선의 물리적 전파를 시뮬레이션하여 사실적인 렌더링을 구현합니다. 래스터화와 달리 반사, 굴절, 전역 조명을 자연스럽게 처리할 수 있습니다.

## 레이 트레이싱 vs 래스터화

| 특성 | 래스터화 | 레이 트레이싱 |
|------|----------|---------------|
| **반사** | 근사 (SSR) | 정확함 |
| **그림자** | Shadow Map | 정확한 Soft Shadow |
| **GI** | Baked/근사 | 실시간 가능 |
| **성능** | 빠름 | 느림 (가속 필요) |
| **복잡도** | O(primitives) | O(rays × primitives) |
