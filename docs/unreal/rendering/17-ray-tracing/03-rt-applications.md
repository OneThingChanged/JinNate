│                                                                 │
│  Perfect Mirror:                                                │
│                                                                 │
│       입사 광선     │     반사 광선                            │
│            ╲        │        ╱                                 │
│             ╲       │       ╱                                  │

│               ╲     │     ╱                                    │
│  ───────────────────●─────────────── 표면                     │

│                                                                 │
