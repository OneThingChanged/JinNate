## 핵심 기술

### Path Tracing

| 기능 | 설명 |
|------|------|
| **글로벌 일루미네이션** | 물리 기반 정확한 간접광 |
| **반사** | 멀티바운스 정확한 반사 |
| **굴절** | 물리적 굴절/분산 |
| **소프트 섀도우** | 정확한 Area Light 그림자 |
| **앰비언트 오클루전** | 자연스러운 접촉 그림자 |

### Movie Render Queue

| 기능 | 설명 |
|------|------|
| **고해상도** | 8K+ 렌더링 |
| **안티앨리어싱** | 높은 샘플 수 |
| **모션 블러** | 정확한 셔터 시뮬레이션 |
| **다중 패스** | AOV 출력 |
| **포맷** | EXR, PNG, ProRes 등 |

### Virtual Production

| 기능 | 설명 |
|------|------|
| **nDisplay** | 멀티 디스플레이 클러스터 |
| **In-Camera VFX** | LED 월 촬영 |
| **Live Link** | 실시간 데이터 스트리밍 |
| **카메라 트래킹** | 실시간 카메라 동기화 |
