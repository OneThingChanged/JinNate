# 벤더별 드라이버

NVIDIA, AMD, Intel 드라이버 특성을 설명합니다.

---

## NVIDIA

### 드라이버 구조

```
┌─────────────────────────────────────────────────────────────────┐
│                    NVIDIA 드라이버                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  GeForce/Quadro 드라이버 (프로프라이어터리)                     │
│  ├── Windows: nvlddmkm.sys (KMD), nvwgf2umx.dll (UMD)          │
│  └── Linux: nvidia.ko (독점 커널 모듈)                         │
│                                                                 │
│  Nouveau (오픈소스, 리버스 엔지니어링)                          │
│  └── 제한된 성능, 최신 GPU 지원 한계                           │
│                                                                 │
│  특수 기능:                                                     │
│  • RT Cores: 하드웨어 레이 트레이싱                            │
│  • Tensor Cores: AI 가속 (DLSS)                                │
│  • NVLink: 다중 GPU 인터커넥트                                 │
│  • CUDA: 범용 GPU 컴퓨팅                                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### NVIDIA 아키텍처 세대

| 아키텍처 | 연도 | 특징 |
|----------|------|------|
| **Pascal** | 2016 | VR Ready |
| **Turing** | 2018 | RT Cores, Tensor Cores |
| **Ampere** | 2020 | 2세대 RT/Tensor |
| **Ada Lovelace** | 2022 | 3세대 RT/Tensor |

## AMD

### 드라이버 구조

```
┌─────────────────────────────────────────────────────────────────┐
│                    AMD 드라이버                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Radeon Software (공식)                                         │
│  ├── Windows: Adrenalin Edition                                │
│  └── Linux: AMDGPU-PRO (하이브리드)                            │
│                                                                 │
│  AMDGPU (오픈소스)                                              │
│  ├── Linux 커널에 포함                                         │
│  ├── Mesa RADV (Vulkan)                                        │
│  └── Mesa RadeonSI (OpenGL)                                    │
│                                                                 │
│  특수 기능:                                                     │
│  • Ray Accelerators: 하드웨어 RT                               │
│  • Infinity Cache: 대용량 L3 캐시                              │
│  • Smart Access Memory: CPU-GPU 메모리 직접 접근              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### AMD 아키텍처

| 아키텍처 | 연도 | 특징 |
|----------|------|------|
| **GCN** | 2012-2019 | Compute 최적화 |
| **RDNA** | 2019 | 게이밍 최적화 |
| **RDNA 2** | 2020 | 하드웨어 RT |
| **RDNA 3** | 2022 | 칩렛 설계 |

## Intel

### 드라이버 구조

```
┌─────────────────────────────────────────────────────────────────┐
│                    Intel 드라이버                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  통합 그래픽 (iGPU)                                             │
│  ├── Windows: 공식 드라이버                                    │
│  └── Linux: i915 (오픈소스, 커널 포함)                         │
│                                                                 │
│  Arc (dGPU)                                                     │
│  └── Xe 아키텍처 기반                                          │
│                                                                 │
│  특징:                                                          │
│  • 오픈소스 우선 정책 (Linux)                                  │
│  • Quick Sync: 하드웨어 비디오 인코딩/디코딩                   │
│  • Xe-HPG: 게이밍 최적화 아키텍처                              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

