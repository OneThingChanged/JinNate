# Ch.16 그래픽 드라이버

그래픽 드라이버의 구조와 동작 원리를 분석합니다.

---

## 개요

그래픽 드라이버는 OS와 GPU 하드웨어 간의 통신을 가능하게 하는 소프트웨어입니다. OS 함수 호출을 디바이스 특화 명령으로 변환하고 렌더링 작업을 위한 저수준 하드웨어 접근을 제공합니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    그래픽 드라이버 계층                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                    Application                           │   │
│  │              (DirectX 12, Vulkan, Metal)                 │   │
│  └───────────────────────┬─────────────────────────────────┘   │
│                          │                                      │
│  ┌───────────────────────┴─────────────────────────────────┐   │
│  │              User-Mode Driver (UMD)                      │   │
│  │         • 명령 검증, 셰이더 컴파일, 리소스 관리         │   │
│  └───────────────────────┬─────────────────────────────────┘   │
│                          │ (System Call)                       │
│  ┌───────────────────────┴─────────────────────────────────┐   │
│  │              Kernel-Mode Driver (KMD)                    │   │
│  │         • 메모리 관리, 스케줄링, 하드웨어 제어          │   │
│  └───────────────────────┬─────────────────────────────────┘   │
│                          │                                      │
│  ┌───────────────────────┴─────────────────────────────────┐   │
│  │                    GPU Hardware                          │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

## 드라이버 핵심 기능

| 기능 | 설명 |
|------|------|
| **모드 설정** | 해상도, 주사율 구성 |
| **메모리 관리** | GPU 메모리 할당/해제 |
| **명령 처리** | 커맨드 큐 관리 |
| **전력 관리** | 클럭, 전압 제어 |
| **보안** | 명령 검증, 메모리 보호 |
