# XR 개념

VR, AR, MR, XR의 정의와 특징을 설명합니다.

---

## VR (Virtual Reality)

완전히 몰입형 3D 환경을 생성합니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    VR 시스템 구성                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                    VR 헤드셋                             │   │
│  │  ┌─────────┐           ┌─────────┐                      │   │
│  │  │ 왼쪽 눈 │           │ 오른쪽 눈│                      │   │
│  │  │ 디스플레이│           │ 디스플레이│                      │   │
│  │  └─────────┘           └─────────┘                      │   │
│  │        ▲                     ▲                          │   │
│  │        │      렌즈           │                          │   │
│  │        └─────────┬───────────┘                          │   │
│  │                  │                                       │   │
│  │             사용자 눈                                    │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  필수 요소:                                                     │
│  • HMD (Head-Mounted Display)                                  │
│  • 모션 트래킹 (6DOF)                                          │
│  • 스테레오 렌더링                                             │
│  • 고프레임률 (90Hz+)                                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 6DOF (Six Degrees of Freedom)

```
┌─────────────────────────────────────────────────────────────────┐
│                    6DOF 추적                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  회전 (Rotation):                  이동 (Translation):         │
│                                                                 │
│       Yaw                              Y (Up/Down)             │
│        │                                   │                    │
│        │   Pitch                           │                    │
│        │  ╱                                │                    │
│        │ ╱                                 │                    │
│        ●───────Roll              X────────●────────Z           │
│                              (Left/Right)  (Forward/Back)       │
│                                                                 │
│  3DOF: 회전만 추적 (모바일 VR)                                 │
│  6DOF: 회전 + 이동 추적 (PC/스탠드얼론 VR)                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## AR (Augmented Reality)

현실 세계에 가상 객체를 오버레이합니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    AR 렌더링                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  입력:                                                          │
│  ┌─────────────┐     ┌─────────────┐                           │
│  │ 카메라 영상 │  +  │ 가상 객체   │                           │
│  │ (현실 세계) │     │ (3D 모델)   │                           │
│  └──────┬──────┘     └──────┬──────┘                           │
│         │                   │                                   │
│         └─────────┬─────────┘                                   │
│                   │                                             │
│                   ▼                                             │
│         ┌─────────────────┐                                    │
│         │    합성 영상    │                                    │
│         │ (현실 + 가상)   │                                    │
│         └─────────────────┘                                    │
│                                                                 │
│  기술 요구사항:                                                 │
│  • 환경 인식 (평면 감지)                                       │
│  • 조명 추정                                                   │
│  • 오클루전 처리                                               │
│  • 카메라 캘리브레이션                                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## MR (Mixed Reality)

현실과 가상 객체가 실시간으로 상호작용합니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    MR 특징                                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  AR과의 차이점:                                                 │
│                                                                 │
│  AR: 가상 객체가 현실 위에 떠 있음                              │
│      (오클루전 없음, 상호작용 제한)                            │
│                                                                 │
│  MR: 가상 객체가 현실과 상호작용                               │
│      • 현실 테이블 위에 가상 객체 배치                         │
│      • 현실 벽 뒤로 가상 객체가 가려짐                         │
│      • 현실 물체와 가상 물체의 충돌                            │
│                                                                 │
│  예시: Microsoft HoloLens                                       │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                                                         │   │
│  │    [현실 테이블]                                       │   │
│  │         │                                               │   │
│  │    ┌────┴────┐                                         │   │
│  │    │ 가상    │  ← 테이블에 고정                        │   │
│  │    │ 객체    │                                         │   │
│  │    └─────────┘                                         │   │
│  │                                                         │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

## XR (Extended Reality)

AR, VR, MR을 포괄하는 용어입니다.

| 기술 | 환경 | 상호작용 | 디바이스 |
|------|------|----------|----------|
| **VR** | 완전 가상 | 가상 객체만 | HMD |
| **AR** | 현실 + 오버레이 | 제한적 | 폰, 글래스 |
| **MR** | 현실-가상 융합 | 양방향 | HMD, 글래스 |
| **XR** | 모두 포괄 | - | - |
