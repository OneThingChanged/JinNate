# Ch.15 XR 렌더링

VR, AR, MR을 포함한 XR (Extended Reality) 렌더링 기술을 분석합니다.

---

## 개요

XR(Extended Reality)은 VR, AR, MR을 포괄하는 용어로, 현실 세계를 증강하거나 대체하는 기술입니다. XR 렌더링은 낮은 지연시간, 높은 프레임률, 스테레오 렌더링 등 특수한 요구사항을 가집니다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    XR 스펙트럼                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  현실 세계 ◀────────────────────────────────────▶ 가상 세계   │
│                                                                 │
│       AR              MR                  VR                   │
│   (증강현실)       (혼합현실)           (가상현실)              │
│                                                                 │
│  ┌─────────┐     ┌─────────┐         ┌─────────┐              │
│  │ 현실 +  │     │현실과   │         │ 완전한  │              │
│  │ 가상    │     │가상의   │         │ 가상    │              │
│  │ 오버레이│     │상호작용 │         │ 환경    │              │
│  └─────────┘     └─────────┘         └─────────┘              │
│                                                                 │
│  예: Pokemon Go   예: HoloLens        예: Quest, PSVR         │
│      Snapchat                                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

## XR 렌더링 요구사항

| 요구사항 | 목표값 | 이유 |
|----------|--------|------|
| **프레임률** | 90fps+ | 멀미 방지 |
| **지연시간** | <20ms | 현실감 유지 |
| **스테레오 렌더링** | 양안 동시 | 입체감 제공 |
| **해상도** | 눈당 2K+ | 선명도 확보 |
| **FOV** | 100°+ | 몰입감 |

```
┌─────────────────────────────────────────────────────────────────┐

│                                                                 │

│  3. Timewarp / Spacewarp                                        │
│     └── 프레임 보간으로 지연시간 보상                          │
│                                                                 │
│  4. 렌즈 왜곡 보정                                              │
│     └── 광학계 왜곡을 소프트웨어로 보정                        │
│                                                                 │

│     └── 단일 드로우 콜로 양안 렌더링                           │

└─────────────────────────────────────────────────────────────────┘
```
