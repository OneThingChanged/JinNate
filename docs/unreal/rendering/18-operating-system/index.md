## 핵심 개념

| 개념 | 설명 |
|------|------|
| Process | 실행 중인 프로그램, 독립된 메모리 공간 |
| Thread | 프로세스 내 실행 단위, 메모리 공유 |
| Context Switch | CPU가 작업을 전환할 때 상태 저장/복원 |
| Scheduling | 어떤 스레드를 언제 실행할지 결정 |
| Synchronization | 스레드 간 협력과 경쟁 관리 |
| Virtual Memory | 물리 메모리를 추상화한 논리 주소 공간 |
