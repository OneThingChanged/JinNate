## 아키텍처 진화

### Fixed Function → Programmable

```
┌─────────────────────────────────────────────────────────────────┐
│                    파이프라인 진화                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Fixed Function (1996-2000):                                    │
│  ┌─────┐   ┌─────┐   ┌─────┐   ┌─────┐   ┌─────┐             │
│  │ T&L │ → │Clip │ → │Rast │ → │ Tex │ → │Blend│             │
│  │Fixed│   │Fixed│   │Fixed│   │Fixed│   │Fixed│             │
│  └─────┘   └─────┘   └─────┘   └─────┘   └─────┘             │
│                                                                 │
│  Programmable (2001+):                                          │
│  ┌──────┐   ┌─────┐   ┌─────┐   ┌──────┐   ┌─────┐           │
│  │  VS  │ → │Clip │ → │Rast │ → │  PS  │ → │Blend│           │
│  │Shader│   │Fixed│   │Fixed│   │Shader│   │Fixed│           │
│  └──────┘   └─────┘   └─────┘   └──────┘   └─────┘           │
│                                                                 │
│  Unified (2006+):                                               │
│  ┌──────────────────────────────────────────────────────────┐  │
│  │              Unified Shader Processors                    │  │
│  │  ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐      │  │
│  │  │ SP │ │ SP │ │ SP │ │ SP │ │ SP │ │ SP │ │ SP │ ...  │  │
│  │  └────┘ └────┘ └────┘ └────┘ └────┘ └────┘ └────┘      │  │
│  │                                                          │  │
│  │  동적으로 VS/GS/PS/CS 역할 분배                         │  │
│  └──────────────────────────────────────────────────────────┘  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
