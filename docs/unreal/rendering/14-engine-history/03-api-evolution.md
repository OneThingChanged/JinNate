## 셰이더 모델 진화

| 버전 | 시기 | 주요 기능 |
|------|------|----------|
| **SM 1.0** | 2001 | 기본 VS/PS, 8개 텍스처 |
| **SM 2.0** | 2002 | 256 명령어, 동적 분기 |
| **SM 3.0** | 2004 | 무제한 명령어, 정점 텍스처 |
| **SM 4.0** | 2006 | Geometry Shader, 정수 연산 |
| **SM 5.0** | 2009 | Tessellation, Compute |
| **SM 6.0** | 2017 | Wave Intrinsics |
| **SM 6.5** | 2020 | Mesh Shader, Ray Tracing |
