## 1990년대 후반 엔진 기술

| 기술 | 설명 |
|------|------|
| **BSP Tree** | 씬 분할, 가시성 계산 |
| **Portal Rendering** | 실내 환경 최적화 |
| **Lightmap** | 사전 계산된 조명 |
| **LOD System** | 거리 기반 디테일 조절 |
| **Particle System** | 빌보드 기반 효과 |
| **Skeletal Animation** | 본 기반 캐릭터 애니메이션 |
