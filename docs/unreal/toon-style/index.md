│  ┌───────────────────────────────────────────────────────────┐ │

│  │ • 애니메이션 스타일                                       │ │

│  │ 후도 스타일 (厚涂 / Impasto Style)                       │ │

│  │ • SSS + Specular 표현                                    │ │
│  │ • 일러스트/유화 스타일                                    │ │

│                                                                 │
