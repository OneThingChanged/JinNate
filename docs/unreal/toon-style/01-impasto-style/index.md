# 후도 스타일 2D 렌더링

> 원문: [厚涂风格实时二次元渲染 시리즈](https://zhuanlan.zhihu.com/p/548788797) by 骨鱼子

후도(厚涂, Impasto) 스타일의 실시간 2차원 캐릭터 렌더링을 다룹니다.

---

## 후도 스타일이란?

```
┌─────────────────────────────────────────────────────────────────┐
│                    厚涂 (Impasto) 스타일                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  후도(厚涂)는 서양 유화에서 유래한 기법입니다.                  │
│  화가가 캔버스에 두꺼운 물감을 붓이나 팔레트 나이프로           │
│  발라서 마치 크림을 바르듯이 표현하는 기법입니다.               │
│                                                                 │
│  특징:                                                          │
│  • 색 블록이 뚜렷함 (두꺼운 물감으로 인한 이산적 그라데이션)   │
│  • 경계가 부드러움 (물감의 혼합으로 인한 자연스러운 전환)       │
│  • 질감 표현이 풍부함                                          │
│                                                                 │
│  셀 셰이딩 vs 후도 스타일:                                      │
│  ┌─────────────────────┬─────────────────────┐                 │
│  │    셀 셰이딩        │    후도 스타일       │                 │
│  │  ┌───────────────┐ │  ┌───────────────┐  │                 │
│  │  │ ████████      │ │  │ ████▓▓▒▒░░   │  │                 │
│  │  │ ░░░░░░░░      │ │  │               │  │                 │
│  │  └───────────────┘ │  └───────────────┘  │                 │
│  │  명확한 경계        │  부드러운 전환       │                 │
│  └─────────────────────┴─────────────────────┘                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 사전 지식

이 시리즈를 시작하기 전에 다음 내용을 이해하고 있어야 합니다:

- 기본 셀 셰이딩 (이진화 조명 모델)
- Outline 렌더링 기법
- Half-Lambert 조명 모델
- Ramp Texture / LUT 사용법
