## 개요

```
┌─────────────────────────────────────────────────────────────────┐
│                    셀 셰이딩이란?                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  "셀 룩", "툰 셰이더"라고도 불리며                              │
│  "3D인데 손으로 그린 것 같은 외관"을 구현하는 기법              │
│                                                                 │
│  ┌─────────────┐         ┌─────────────┐                       │
│  │   3D 모델   │   →     │  애니메이션  │                       │
│  │  (입체적)   │  변환    │   스타일    │                       │
│  └─────────────┘         └─────────────┘                       │
│                                                                 │
│  특징:                                                          │
│  • 색 수를 줄여 모델의 입체감 조절                              │
│  • 아웃라인(주선) 추가                                          │
│  • 애니메이션/만화풍 표현                                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

## 구현 방법 3가지

| 방법 | 설명 | 난이도 |
|------|------|--------|
| **Unlit 셰이더** | 머티리얼에서 직접 라이팅 계산 | 쉬움 |
| **포스트 이펙트** | 화면 전체에 효과 적용 | 보통 |
| **엔진 개조** | 렌더링 파이프라인 수정 | 어려움 |

이 시리즈에서는 **Unlit 셰이더** 방식을 사용합니다.
