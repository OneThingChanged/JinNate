## 모델 비교

| 기능 | Qwen3-TTS | GPT-SoVITS |
|------|-----------|------------|
| **내장 화자** | 9명 (Sohee 등) | 없음 |
| **음성 복제** | 3초 샘플 | 1분 샘플 (고품질) |
| **파인튜닝** | 어려움 | WebUI로 간편 |
| **설치 크기** | ~4GB | ~8GB |
| **VRAM** | 6GB+ | 4GB+ |
| **추천 용도** | 빠른 TTS | 커스텀 음성 제작 |
