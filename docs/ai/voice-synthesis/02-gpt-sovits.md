## 사용 방법

### Step 1: 참조 오디오 준비

- **1분 이상**의 깨끗한 음성 파일 (WAV 권장)
- 배경 소음 없는 음성
- 한 명의 화자만 포함

### Step 2: WebUI에서 학습

```
┌─────────────────────────────────────────────────────────────────┐
│                      학습 워크플로우                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌────────────────┐                                             │
│  │ 0-음성 슬라이싱 │  오디오를 짧은 클립으로 분할               │
│  └───────┬────────┘                                             │
│          │                                                      │
│          ▼                                                      │
│  ┌────────────────┐                                             │
│  │ 1-ASR          │  음성을 텍스트로 변환 (자동 전사)           │
│  └───────┬────────┘                                             │
│          │                                                      │
│          ▼                                                      │
│  ┌────────────────┐                                             │
│  │ 2-학습         │  GPT/SoVITS 모델 파인튜닝                   │
│  └────────────────┘                                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

1. **0-음성 슬라이싱**: 오디오를 짧은 클립으로 분할
2. **1-ASR**: 음성을 텍스트로 변환
3. **2-학습**: GPT/SoVITS 모델 파인튜닝

### Step 3: 추론 (음성 생성)

1. **1C-추론** 탭 이동
2. 학습된 모델 선택
3. 텍스트 입력 후 음성 생성

## Qwen3-TTS와 비교

| 항목 | GPT-SoVITS | Qwen3-TTS |
|------|------------|-----------|
| **내장 화자** | 없음 | 9명 |
