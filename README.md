## 로컬 실행

```bash
# 의존성 설치
pip install -r requirements.txt

# 개발 서버 실행
mkdocs serve

# 브라우저에서 http://127.0.0.1:8000 접속
```

## 배포

`main` 브랜치에 푸시하면 GitHub Actions가 자동으로 빌드하고 GitHub Pages에 배포합니다.
